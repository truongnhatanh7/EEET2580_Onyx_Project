Tutorial 1: Log in to the application
  1. From the login page, type your registered username in the Username field.
  2. Type the registered password in the Password field.
  3. Select the Log in button.

In contrast, if the user has not registered for the account, select the signup button and fill in all the fields of the registration form then click the Sign up button for logging in.

Tutorial 2: Set up your project
  1. At dashboard page, select + in Your Project area
  2. Enter a new name for your workspace.
  3. Click at the newly created workspace.
  4. You can now create a new task list for your project.
Tutorial 3: Create the workspace list
  1. From the workspace, select the Create new list under the workspace's name./li>
  2. Type the name in the Enter new list name field.
  3. Click the Create button for creating a new list.
Tutorial 4: Add a new task within the workspace list
  1. From the list, select the Add task button.
  2. Type the name in the Enter new task field.
  3. Click the Create button for creating a new task.
Tutorial 5: Edit a task within the workspace list
  1. From the list, find the task you want to make a change to and select the Edit button.
  2. Type the new name for the task.
  3. Click the Save button for saving the new name.
Tutorial 6: Drag and Drop a task to another list

From the list, find the task you want to move to and hold a click on it. Then, drag the task to the list where you want to place it and drop a task into it.

Tutorial 7: Delete task within the list
  1. From the list, find the task you want to make a change to and select the Edit button.
  2. Click the Delete task button for deleting the task.
Tutorial 8: Delete the list within the workspace

From the workspace, find the list you want to delete and click the X button in the top right corner.

Tutorial 9: Work with collaborators within the workspace
  1. From the workspace, select the avatar icon in the top right corner.
  2. Select the Project status.
  3. In the Project status dialogue, type the username of the collaborators that you want to add.
  4. Select Add button for adding a new collaborator.

    5. In contrast, click the X button next to the collaborator's name to remove them from the project.

Tutorial 10: Delete your project
  1. From the workspace, select the avatar icon in the top right corner.
  2. Select the Project status.
  3. Select the Delete this workspace button.
  4. In the Danger Zone field, type delete this workspace to confirm the deletion.
Tutorial 11: Sign out of the project

From the dashboard, select the avatar icon in the top right corner. Select the Sign-out button for logging out of the project.
